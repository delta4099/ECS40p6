# myshell

myshell (also called funix) is a shell with the following commands: 

```
cd - changes the directory 

cdmod

chown

cp 

exit

ls [-l] 

mkdir

su 

umask

touch 

```
### Compile from source

```
git clone https://github.com/michael-a-wong/myshell.git

cd /myshell/source

make clean

make

./funix.out
```
